# P&B Comunicação Visual - Apresentação

Este repositório contém os arquivos da apresentação da P&B Comunicação Visual.

## Arquivos

- `apresentacao.html`: apresentação principal em HTML.
- `prompts-imagens.md`: prompts utilizados para geração de imagens.

## Como visualizar

1. Baixe/clone o repositório.
2. Abra o arquivo `apresentacao.html` no navegador.
